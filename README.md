## 📖 How to Use

### Quick Start (3 Simple Steps)

<div align="center">

**1️⃣ Download** → **2️⃣ Run** → **3️⃣ Extract Audio**

</div>

<br>

### Detailed Instructions

1. **📥 Download the Application**
   - Click the download button above
   - Save `HikariMP3Downloader.exe` to your computer
   - No installation needed!

2. **▶️ Run the Application**
   - Double-click `HikariMP3Downloader.exe`
   - Windows may show a security warning (click "More info" → "Run anyway")
   - The application will open immediately

3. **📋 Paste YouTube URL**
   - Copy a YouTube video URL
   - Paste it into the input field
   - Click "Verify" to preview the video

4. **⚙️ Configure Settings** (Optional)
   - Choose audio quality (128-320 kbps)
   - Select processing engine (yt-dlp recommended)
   - Choose conversion method (FFmpeg recommended)
   - Select output folder

5. **🎵 Extract Audio**
   - Click "Extract Audio" button
   - Wait for processing (usually 10-30 seconds)
   - Your MP3 will be saved to the output folder

6. **✅ Enjoy Your MP3**
   - Click "Open Folder" to find your file
   - File is named: `[Video Title].mp3`

## 🙋 Frequently Asked Questions

<details>
<summary><b>❓ Do I need to install Python or any dependencies?</b></summary>

<br>

**No!** Hikari Mp3 Downloader is a standalone executable.

- ✅ Everything is included in the .exe
- ✅ No Python installation needed
- ✅ No pip or dependencies
- ✅ Just download and run

**Optional:** FFmpeg for best quality (but MoviePy works without it)

</details>

<details>
<summary><b>❓ Why is my download failing?</b></summary>

<br>

**Common solutions:**

1. ✅ **Update libraries** - Click "🔄 Update Libraries" button
2. ✅ **Try different engine** - Switch between yt-dlp and pytube
3. ✅ **Check URL format** - Remove playlist parameters (`&list=`)
4. ✅ **Verify internet** - Ensure stable connection
5. ✅ **Test with different video** - Try a short, public video first

**If still failing:**
- Video may be private, deleted, or restricted
- Age-restricted content may not work
- Geographic restrictions may apply
- Try again later (YouTube may be blocking temporarily)

</details>

<details>
<summary><b>❓ Do I need FFmpeg?</b></summary>

<br>

**No, but it's recommended for best results:**

**Without FFmpeg:**
- ✅ App works perfectly
- ✅ Uses MoviePy converter
- ⚠️ Slower conversion
- ⚠️ Slightly lower quality

**With FFmpeg:**
- ✅ Faster conversion (3-5x speed)
- ✅ Best quality preservation
- ✅ Industry-standard processing
- 📥 Download from [ffmpeg.org](https://ffmpeg.org/download.html)

**Installation:**
1. Download FFmpeg
2. Extract to `C:\ffmpeg`
3. Add `C:\ffmpeg\bin` to system PATH
4. Restart Hikari

</details>
